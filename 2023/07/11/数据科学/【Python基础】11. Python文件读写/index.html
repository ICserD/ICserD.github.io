文本文件读写

打开文件

open打开文件后，f就在程序中代表这个文件。

  • f.readlines读取文件的内容，内容存放在列表里面，列表的每个元素都是一个字符串，代表文件的一行

  • open之后返回一个可迭代的文件对象给f

  • 打开模式可选

    访问模式 说明
    r 默认，以只读方式打开
    w 打开一个文件只用于写入
    a 追加
    rb 二进制只读
    wb 二进制写
    ab 二进制追加

写文件

open会在目录创建文件（如果文件不存在），但是如果目录不存在，则会报错。因为open不能创建一个文件夹。

  • 常见的文件写入函数

    方法 描述
    f.write(s) 向文件写入一个字符串或字节流
    f.writelines(lines) 将一个元素为字符串的列表写入文件
    f.seek(offset) 改变当前文件操作指针的位置，offset的值：
    0：文件开头；1：当前位置；2：文件结尾

二进制文件

  • 数据库文件、图像文件、可执行文件、音频文件、OFFICE文档均属于二进制文件

  • 二进制文件不能使用记事本或其他文本文件编辑软件正常读写，也无法直接通过Python进行直接解析；

    必须正确理解二进制文件结构和序列化规则，才能准确理解二进制文件内容并设计正确的反序列化规则；

  • 序列化：把内存中的数据在不丢失其类型信息的情况下转成对象的二进制形式的过程，对象序列化后的形式经过正确的反序列化可以准确无误地恢复为原有的对象；

  • Python中常用的序列化模块主要有：pickle，struct，marshal

举例：用pickle模块实现内存对象的保存

步骤：

  • 序列化：将对象在内存中的数据直接抓取出来，转换成有序的文本

    pikle包的dumps()方法可以将对象转成字符串的形式
