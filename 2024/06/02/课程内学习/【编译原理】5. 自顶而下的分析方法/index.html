小结：递归下降分析方法

自上而下的分析方法，也称递归子程序下降的分析方法解决问题；

  1. 把问题的核心用文法规则的方式表达出来；
  2. 然后利用递归下降的算法设计方法，也就是递归子程序的方法组织实现；
    • 一个非终结符号就是一个函数
    • 遇到终结符号作匹配，非终结符号作函数调用

递归则程序的问题分析

递归子程序的方法固然好用，但是针对不同特殊类型的文法规则仍然会出现一些问题，所以接下来我们分享不同的经典的文法规则，用递归子程序的方法实现其功能，纠正不同的问题，使得递归下降的分析方法更加完善；

左工因子问题

例子：写出下面的文法的递归子程序$$\displaylines{if-stmt\ \to\ if\ (exp)\ statement\ |\ if\ (exp)\ statement\ else\ statement}$$问题分析：

  1. 由于该文法规则的推导有两种情况，所以需要分支语句；
  2. 但是由于两种情况的开头第一个符号都是相同的，都是if（非终结符号），所以对应着同一个函数或者终结符号，那么用前面的方法进行匹配，无法区分什么时候选第一条规则，什么时候选择第二条规则；

解决方法：

  1. 方法一：提公因式，把规则进行改写：if−stmt → if (exp) statement (ϵ∣ else statement)if-stmt\ \to \ if\ (exp) \ statement\ (\epsilon |\ else\ statement)if−stmt → if (exp) statement (ϵ∣ else statement)
  2. 方法二：用扩充的BNF改造原来的文法规则：if−stmt → if (exp) statement [else statement]if-stmt\ \to \ if\ (exp) \ statement\ [else\ statement]if−stmt → if (exp) statement [else statement]

问题归纳：左公因子问题
当两个或多个文法规则共享一个通用的公共前缀时，如A→αβ∣αγ∣αβ1∣...A\to \alpha\beta|\alpha\gamma|\alpha\beta1|...A→αβ∣αγ∣αβ1∣...等
我们需要将公共的前缀，也就是左公因子提取出来；
比如：
A→ αβ∣αγA \to \ \alpha\beta|\alpha\gammaA→ αβ∣αγ可以进行公因子提取；
结果为：
A→α(β∣γ)A \to \alpha(\beta|\gamma)A→α(β∣γ)
或者可以在分解后，将该规则分解为两个规则(==一般在提取公因子后，非公因子部分很长的时候采用这种方法==)：

\displaylines{A \to \alpha A^,\\ A^, \to \beta|\gamma }

例题：写出下面文法的递归子程序$$\displaylines{G[S]={S \to SaAB | \epsilon\
A \to cdA \
B \to efB|f
}}$$
问题分析：
以S→SaS\to SaS→Sa为例，对含有左递归的表达式，在用递归下降子程序实现时，会出现函数调用死递归；

解决方法：
经过归约分析，我们对S→SaAB∣ϵS \to SaAB|\epsilonS→SaAB∣ϵ这个表达式的功能进行分析，实际上就是可以实现aAB的重复出现；于是我们采用扩充的BNF改写文法规则S→{aAB}S \to \{aAB\}S→{aAB}

也可以改为右递归，来避免死递归的出现；
由于该文法实现的是aAB重复出现的功能，我们使用右递归也能实现，我们如何将该表达式转化为右递归呢？

\displaylines{S\to SaAB|\epsilon\\ \therefore S \to T^,\\ T^, \to aABT|\epsilon \ \ \\这一步实现了aAB的重复 \\ \therefore S \to aABS|\epsilon \\ 这一步进行合并，原文法变为右递归的形式 }

间接左递归

例子：研究下面的文法$$\displaylines{A \to Aa|Bb|Cc \
B \to Ab|Bb\
C \to Ac|Cb
}$$
问题分析：上面的例子中，如果把B带入第一条文法，又会出现A的左递归，这种形式的文法称为间接左递归；

解决方法：

\displaylines{A \to Aa|Bb|Cc \\ \therefore A \to (Bb|Cc)A^, \to BbA^,|CcA^,,\\ A^, \to aA^,|\epsilon \\处理完A的左递归之后，解决B的时候带入A的结果即可\\ B \to Ab|Bb\\ B \to BbA^,b|CcA^,b|Bb \\ 于是这时候B从间接左递归变成了直接左递归\\ \therefore B \to B(bA^,b|b)|CcA^,b \\ ... }

问题归纳：解决间接左递归

  1. 将文法的非终结符号按任意一种方式排序
  2. 逐个逐个非终结符号进行消除直接左递归，得到干净的非终结符号
  3. 将干净非终结符号带入未解决的非终结符中，并将其消除干净
  4. 重述上述过程

First集合的求解

在递归下降程序中，为了提高分析效率，往往要求出每一条文法规则的开头的非终结符号的所有打头的终结符号，也称为First集合。这样可以避免回溯性。
不过需要注意的是First集合不一定只有非终结元素，还有可能有ϵ\epsilonϵ空串。
例题：现有文法$$\displaylines{S \to ABC | D\
A \to aB | \epsilon \
B \to cC|\epsilon \
C \to eC | \epsilon\
D \to i|j
}$$
求First(D)、First(A)、First(S)。

小结：

  1. 求First集合元素是为了避免递归下降子程序的回溯导致效率低，所以我们在==求First集合元素之前，应该将文法规则进行处理，消除左递归、提左公因子等操作==！
  2. 若文法规则多条，分别求first集合元素
  3. 如果first元素是非终结符号，求其终结符号first元素
  4. 如果first元素是ϵ\epsilonϵ，求下一个符号的first集合
  5. 如果某条文法规则全部都含有ϵ\epsilonϵ，类似于S→ABCS \to ABCS→ABC，于是可以将ϵ\epsilonϵ加入first集合
